# 🚀 Lead Generation Landing Page

A modern and responsive **Lead Generation Landing Page** designed for digital marketing services. This project helps businesses attract potential clients using high-converting UI, clear call-to-actions, and a contact form integration.

## 📌 Features

- ✅ Fully Responsive Design (Mobile, Tablet, Desktop)
- ✅ Modern UI/UX with smooth animations
- ✅ Sticky Navigation Bar with active link highlight
- ✅ Hero Section with Call-to-Action buttons
- ✅ Services & Business Solutions Section
- ✅ Pricing / Offer Section
- ✅ Why Choose Us Section
- ✅ Results / Expectations Section
- ✅ Contact Form (Google Sheets Integration Ready)
- ✅ WhatsApp Direct Chat Option
- ✅ Scroll-to-Top Button
- ✅ Smooth Scrolling Navigation


## 🛠️ Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling & Responsive Design
- **JavaScript (Vanilla JS)** – Interactivity & Animations
- **Font Awesome** – Icons
- **Google Fonts** – Typography


## 📂 Project Structure
📁 project-folder
│── index.html
│── style.css
│── script.js
│── images/


## ⚙️ How It Works

- Users land on the page and explore services
- CTA buttons encourage users to take action
- Contact form collects user data
- Data can be sent to **Google Sheets** using Google Apps Script (API URL needs to be added in `script.js`)
